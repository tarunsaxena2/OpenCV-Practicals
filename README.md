# OpenCV Practicals

## About

This project contains 12 basic OpenCV practicals written in Python.

In this project, I learned how to perform different image and video processing operations using OpenCV.

## Libraries Used

* Python
* OpenCV (cv2)
* NumPy

## Practicals

1. Read, Display and Save Image
2. Resize Image
3. Flip Image
4. Draw Shapes and Add Text
5. Image Translation
6. Image Rotation
7. Binary Thresholding
8. Gaussian Blur and Median Blur
9. Morphological Operations (Top Hat and Black Hat)
10. Canny Edge Detection
11. Video Read and Write
12. Webcam Capture

