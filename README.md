# OpenCVPracticals# OpenCV Practicals - Tarun

## Project Description

This project contains basic image and video processing practicals implemented in Python using the OpenCV library.

The assignment demonstrates common OpenCV operations such as reading images, resizing, flipping, drawing shapes, image transformations, thresholding, blurring, morphological operations, edge detection, video processing, and webcam capture.

---

## Technologies Used

* Python 3
* OpenCV (cv2)
* NumPy

---

## Installation

Install the required libraries using:

```bash
pip install opencv-python numpy
```

---

## How to Run

Open the project folder in VS Code and run any practical using:

```bash
python practicals/<file_name>.py
```

Example:

```bash
python practicals/01_read_display_save.py
```

---

## Expected Output

Each practical performs the required OpenCV operation and displays the output using OpenCV windows.

The generated output images and videos are automatically saved in the **outputs** folder.

Screenshots of all practicals are included in the PDF file.
